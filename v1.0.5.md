# The Forge v1.0.5

Release date: 2026-04-30

## Fixed

- Added splash screen coverage for Blender `5.x`, `4.x`, `3.x`, and common `2.x` branches.
- Added major-version fallback splash images for future `5.x` and `4.x` builds.
- Fixed installed versions showing only gradient backgrounds when the branch was not `4.5` or `4.2`.

## Notes

- Splash images are loaded from Wikimedia Commons through file redirect URLs.
- If a specific historical splash file is unavailable remotely, the app keeps the existing gradient fallback.
