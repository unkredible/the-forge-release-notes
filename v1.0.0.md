# The Forge v1.0.0

Release date: 2026-04-28

## Highlights

- Initial public release of The Forge.
- Standalone desktop app for managing Blender versions and project presets.
- Release catalog sourced from the official Blender release servers and daily build catalog.
- Local Blender library with install, uninstall, launch, and version selection.
- Preference and add-on copy/paste flow between installed Blender versions.
- Project library with Markdown-based presets and custom preset import.
- Downloads page with queued install jobs and visible progress.
- Free/Pro structure with Gumroad license activation.
- Update notification channel.

## Notes

- Windows build is the primary tested target for this release.
- macOS and Linux builds are generated by CI but still considered work in progress.
