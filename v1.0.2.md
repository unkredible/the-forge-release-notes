# The Forge v1.0.2

Release date: 2026-04-29

## Added

- Owner license bypass support for internal Pro activation.
- Stronger Supabase-backed update manifest integration.
- Per-version background action feedback while installs, deletes, and preference actions run.

## Changed

- Simplified the settings layout.
- Removed the sidebar local builds statistic.
- Tightened Free/Pro feature gating.
- Hid Pro-only project navigation and advanced settings from Free mode.
- Limited Free release catalog visibility to Official and LTS builds.
- Sorted installed Blender versions from newest to oldest.
- Normalized the license endpoint to the cloud Supabase function.

## Fixed

- Improved update download flow and manual update handling.
- Surfaced download failures more clearly.
- Hardened Blender download URL resolution to reduce stale or invalid catalog links.
- Restricted `.blend` drag and drop to Pro mode.

## Security

- Relaxed only the client-side license key format check while keeping server verification authoritative.
- Removed local debug/development license assumptions from the user-facing flow.
