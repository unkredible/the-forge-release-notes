# The Forge v1.0.3

Release date: 2026-04-29

## Fixed

- Fixed Blender update retry when the target version is already present after a partial update.
- Improved Windows uninstall handling for read-only files and locked folders.
- Refreshed the installed versions list after partial update failures.
- Kept download URL validation and catalog hardening from `v1.0.2`.

## Notes

- This release is focused on reliability during update, uninstall, and recovery flows.
